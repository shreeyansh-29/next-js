What is Next.js
The react framework for building production ready applications for web.
React =>
1. Not quite possible to build a full feature rich application ready to be deployed for production.
2. React is a library for building user interfaces.
3. You have to make decisions on other features of the app like routing, styling, authentication etc.

Next.js => React framework.
1. A package that uses React for building UI.
2. Loaded with a lot more features that enable you to build full fledged production ready apps. 
   Features exactly like routing, styling, authentication, bundle optimization etc.
3. There's no need to install additional packages. Next.js provides everything for you.
4. Does have opinions and conventions need to be followed to implement the above said features.

Why learn Next.js?
Next.js simplifies the process of building a react app for production.

1. File base routing => For adding routes in reactjs, we need to install 3-party lib, configure it and add code everytime to add routes.
2. Pre-rendering => Generates html for each page in advance instead of being done by client side JS. It can result in better performance.
3. API routes => Can creates api.
4. Support for CSS modules (Tailwind.css, styled-components)
5. Authentication
6. Dev and Prod build system
